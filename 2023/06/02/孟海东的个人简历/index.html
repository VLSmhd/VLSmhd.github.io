项目经历

优品抢购 2023年01月——2023年04月

项目技术栈：Springboot + mysql + mybatis-plus + redis + RabbitMQ + Hutool + Lombok

项目描述：优品抢购，为用户搜集附近的各种类型的商铺信息，能让用户快速查询，并且为用户提供了关注、发文等社交系统服务，内置秒杀系统，为商户优惠券秒杀活动提供支持，为用户提供高可用、高性能的秒杀服务。

核心内容：

  • 基于redis 实现共享session登录、登录状态刷新问题
  • 实现查询缓存、数据冷热分离，将热数据缓存到redis中，并实现了良好的缓存更新策略，解决了缓存穿透、缓存击穿等问题
  • 实现优惠券秒杀系统，利⽤乐观锁解决超卖问题、使⽤redis分布式锁redission，lua脚本，实现一人一单功能
  • 基于redis的bitmap、geo等数据结构，实现用户签到、 附近商铺、发布点评⽂章、用户点赞、用户关注等功能
  • 基于feed流的Timeline模式实现好友关注的消息推送服务

技术难点、拓展

  • 基于函数式编程和泛型编写redis对应springboot的工具包
  • 基于redis + RabbitMQ异步解耦，实现异步订单处理
  • 基于redis的Zset数据结构，采用滑动窗口算法编写自定义限流AOP实现秒杀接口限流、防刷
  • 实现feed流消息推送的分页滚动展示效果，利用贪心算法解决时间戳相同的情况下，如何不漏掉消息、避免重复查询消息

发现工大 2023年05月——2023年07月

项目技术栈：SpringCloud、SpringBoot、Mybatis-plus、Mysql、redis、mongoDB、kafka、minIO、xxl-job、jenkins、Elasticsearch、阿里云内容审核接口、nginx

项目描述：发现工大，旨在为校园内的同学提供快捷的校园资讯、最新校园快报。本项目从三端入手：用户端、自媒体端、后台管理端。融合了springcloud微服务技术栈，实现了发布热点文章、文章自动审核、延迟发布任务、文章快速检索、历史记录搜索、热文章计算等。后端项目采用jenkins持续集成工具进行部署。

核心内容：

  • 针对热点文章，使用Freemaker视图渲染模板引擎，实现页面静态化，并部署到nginx，降低对服务器的压力
  • 实现文章自动审核，基于DFA算法实现文章敏感词过滤、借助阿里云内容安全服务审核文本内容，并设置了文章审核服务降级处理策略，减少审核服务压力
  • 全局检索服务：基于Elasticsearch实现基于索引快速查找具体文章，基于MongoDB保存文章搜索记录，快速检索历史记录
  • 文章热度计算，基于KafkaStream实时计算文章点赞及阅读量

技术难点、拓展

  • 基于Kafka的TimeWheel时间轮算法，实现延迟队列，保证文章定时发布
  • 使用OCR技术将图片中的文字提取，录入文本信息进行审核

抖声 2023年08月——2023年09月

项目技术栈：go、gin、gorm、redis、mysql、oss

项目描述：本项目原型基于抖音，在后端实现了基础功能如：登录注册、视频发布、视频审核、点赞、评论、首页浏览视频、feed流推送等。

核心内容——后端视频模块： - 视频上传：app端用户调用视频上传接口，后端处理请求，将视频文件以byte流或文件流形式传输到阿里云oss对象存储中 - 视频feed流推送：基于feed流的timeline模式推送关注视频 - 视频审核：利用GoFFmpeg提取视频关键帧，借助阿里云审核接口进行内容安全审核 - 音频审核：利用audio解码视频中的音频文件，借助语音识别工具将音频文本化，上传到阿里云文本审核进行内容安全审核
