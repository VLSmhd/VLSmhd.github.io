动归规律总结

关键步骤：

  1. 思考子问题的构造。
    1. 一是定义目标问题为子问题
    2. 二是定义非目标问题为子问题
  2. 定义dp数组、初始化dp数组、处理边界问题
  3. 分析子问题状态转移，推导递推方程。（推导子问题有几种状态）

核心：保证最终问题是由一个个子问题解决的即可。

Kadane算法

专门解决子数组最大和的一种算法，在动态规划的基础上优化了空间、时间。

给定数组arr，求解最大子数组和的动归思路：

  1. 定义子问题：要求maxSubSum(n)，其子问题为maxSubSum(i)定义为以i为终止下标的子数组之和的最大值
  2. 定义dp，dp[0] = arr[0]
  3. 递推方程
    • 当dp[i-1] < 0 且 arr[i] > / < 0，则 dp[i] = arr[i]
    • 当dp[i-1] > 0 ，则dp[i] = dp[i-1] + array[i]

kadane算法在基础上优化，使用一个指针保存以 i 为结尾的子数组之和的最大值，使用另一根指针保存迄今为止的子数组最大值之和。

  • 第一个指针：curMax，当前子数组的最大和
  • 第二个指针：globalMax，全局子数组最大和

最高有效位

一个整数 i 如果是2的n次幂，那他只有最高位为1。那我们把 \([2^n , 2^{n+1}]\)里的数x，减去$ 2 ^ n$ ，这样退阶到子问题阶层，然后实际1的个数比这个子问题多一（因为最高位基于 $ 2 ^ n$)

判断整数是否为2的n次幂的方法： i & (i - 1) == 0

环形数组 意味着数组的末端将会与开头相连呈环状。形式上， nums[i] 的下一个元素是 nums[(i + 1) % n] ， nums[i] 的前一个元素是 nums[(i - 1 + n) % n] 。

子数组 最多只能包含固定缓冲区 nums 中的每个元素一次。形式上，对于子数组 nums[i], nums[i + 1], ..., nums[j] ，不存在 i <= k1, k2 <= j 其中 k1 % n == k2 % n 。

最大、最小子数组和（逆向思维）

求出最大最小子数组和，

如果最小子数组和 > 0那就说明整个数组中的元素都是> 0的，返回整个数组的和即可。

如果最小子数组和 < 0，说明数组中存在负数，那就要避开这些负数，但由于数组是环形数组，会分为以下情况：

  • 最大区间在数组末尾和数组开头，形成环路
    • 解决：只要找到最小和区间数组，拿sum - min即可
  • 最大区间就在中间
    • 直接就是max
