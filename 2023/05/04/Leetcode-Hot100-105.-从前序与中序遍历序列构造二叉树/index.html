105. 从前序与中序遍历序列构造二叉树

给定两个整数数组 preorder 和 inorder ，其中 preorder 是二叉树的先序遍历， inorder 是同一棵树的中序遍历，请构造二叉树并返回其根节点。

提示:

  • 1 <= preorder.length <= 3000

  • inorder.length == preorder.length

  • -3000 <= preorder[i], inorder[i] <= 3000

  • preorder 和 inorder 均 无重复 元素

  • inorder 均出现在 preorder

  • preorder 保证 为二叉树的前序遍历序列

  • inorder 保证 为二叉树的中序遍历序列

算法分析：

前序遍历的结点顺序：根、左、右。

中序遍历的结点顺序：左、根、右。

哈希表 + 递归分治：

我们根据前序遍历找到根结点，然后在中序遍历数组中找到对应的位置i，这样i左边就是这颗树的左子树，i右边就是这棵树的右子树。可以分治递归的去构造左右子树。

分治递归：

采用双指针限定数组范围，传入递归函数作为参数。

  • 终止条件，左指针越过右指针
  • 递推逻辑：确定根结点，左右递归确定左右子节点。

时间优化方案：

在中序遍历数组中定位根结点的时候，要嵌套循环，我们可以先用哈希表存储中序遍历中每个元素及其对应的下标，这样在递归过程中能直接以O(1)的时间复杂度定位根节点。
