  1. 滑动窗口大小为1，乘积是1，1 < 10，因此需要加入结果集，res += end - start + 1 = 0 - 0 + 1 = 1 [1];

证明：

假设链表A长度为m，链表B长度为n

    1. 如果a == b：那两个指针会同时指向相交结点，循环结束，返回p || q
    2. 如果a ！= b：p指针会走： a + c + b距离 q指针会走 b + c + a距离，两者走的距离相同，因此（第二次交叉遍历）会到达相交结点
  2. 假如两个链表不相交：设相交部分链表长度为c，A不相交部分为a，b不相交部分为b，那按照遍历思路：
    1. a == b：同时走到null，返回
    2. a ！= b：交叉后，会同时走 m + n个距离，走向null，返回
