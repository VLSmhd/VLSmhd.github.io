15. 三数之和

给你一个整数数组 nums ，判断是否存在三元组 [nums[i], nums[j], nums[k]] 满足 i != j、i != k 且 j != k ，同时还满足 nums[i] + nums[j] + nums[k] == 0 。

请你返回所有和为 0 且不重复的三元组。

注意：答案中不可以包含重复的三元组。

算法分析：

  • 暴力法：

    三重遍历，时间复杂度为O（$ n^3$）。

    代码：略

  • 双指针 + 排序：

    遇事不决，先排序。然后设定一个指针k在最小处，这个指针要在最外层循环中。

    • 外层循环：

      • 剪枝： nums[k] > 0，因为数组已经排过序，所以接下来的和不可能等于0.直接break。
      • K去重：k>0 && nums[k] == nums[k-1] ，说明上一轮循环中，这个值已经用光了，排除。直接continue。

    • 内层循环：双指针i，j。 左指针i在K后面一个位置，右指针j在数组末尾。开始遍历，记录这三个指针所指的数的和sum。

      如果sum＞0，说明 j 所指的数字过大，需要往左移动，注意j指针的去重操作

      如果sum < 0，说明 i 所指的数字过小，需要往右移动，注意去重。

      如果sum = 0，加入到结果集中，此时两个指针都要进行去重操作

      • 去重：当指针所指的元素与下一个（上一个）元素相同的时候，指针继续向前走，直到遇见下一个新元素。本题的写法借鉴了K神的思路，直接在while的循环操作里完成了去重操作。 while(i < j && nums[i] == nums[++i])； 在i到达相同元素集的最后一个元素时，该while还会循环判断一次，这时候i正好跳出相同元素集，到达下一个新的元素

知识补充：

  • Java工具类：java.util.Arrays

    这是一个与数组相关的工具类，里面提供了大量静态方法，用来实现数组常见的操作。

    • 常用方法：

      • 排序数组：

        Arrays.sort( 数组 );对数组默认升序重排序。

        • 排序数字：默认升序
        • 排序字母：不设置排序规则，则先大写后小写。按照字母表顺序。
        • 严格按字母表顺序排序，忽略大小写排序： Arrays.sort(strArray, String.CASE_INSENSITIVE_ORDER)
        • 反向排序：Arrays.sort(Array, Collections.reverseOrder());

      • 打印输出：

        Arrays.toString();

      • 比较两个数组的元素是否相同：

        Arrays.equals();

      • 截取数组：

        Arrays.copeOf(数组， int 截取的元素个数) 和Arrays.copeOfRange(数组， int 起始下标， int 结束下标);

      • 赋值、替换原来的元素：

        Arrays.fill(数组, value)

      • 转为集合：

        static List asList(T... a)
