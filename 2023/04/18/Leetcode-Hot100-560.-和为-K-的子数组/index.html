前缀和：

初学前缀和：本题是我第一次接触这个概念，我总结了前缀和的几个关键点：

给出一个数组a[i] , 前缀和数组s[i]. s[i] = a[1]+a[2]+...+a[i]; s[i]其含义就是数组a[i]的前i项之和。前缀和数组s[i]简单赋值语句如下：

如何得到s[i] = s[i - 1] + a[i];：

s[i] = a[1] + ... + a[i];

s[i-1] = a[1] + ... +a[i-1];

将上面两式子相减得： s[i] - s[i-1] = a[i];

因此s[i] = s[i-1] + a[i];

前缀和的应用：

  • 以O(1)的时间复杂度，得出数组某个区间的和。

    假设数组a[i] ，其前缀和数组s[i]，现在要求区间[L,R]的数组元素之和。(\(0 \leq L \leq R \leq i\))

    根据前缀和数组：

    s[L - 1] = a[0] + ... + a[l - 1];

    s[R] = a[0] + ... + a[l] + ... + a[r];

    上面两式子相减得：s[R] - s[L - 1] = Sum(l,r);
