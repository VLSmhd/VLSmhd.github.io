常见使用问题

OpenEvent

OpenEvent("Global_reload_13956") failed (2: The system cannot find the file specified)

  • 原因：nginx没启动就使用nginx -s reload

nginx进程过多

写工大头条项目的时候，自媒体端怎么都登不进去，后端测试也正常，怀疑是nginx出的问题，果不其然。

原因：进程运行太久，太多，缓存过多。

解决：

打开windows资源管理器，把所有nginx的进程关了 重开。
